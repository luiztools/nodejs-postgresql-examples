# crud-sequelize-postgresql
Fontes do tutorial de CRUD com Node.js, Sequelize e PostgreSQL.

## How to Run

1. create the PG database
2. git clone the repo
3. npm install
4. copy .env.example as .env
5. fill .env variables
6. npm start
