# nodejs-postgresql
A simple CRUD in Node.js with PostgreSQL.

## How to Run
1. create the PG database
2. create the table 'clientes' (SQL available at create-table.sql)
3. git clone the repo
4. npm install
5. copy .env.example as .env
6. fill .env variables
7. npm start
